# CVL Splits

Splits for the CVL Database (Kleber, Fiel, Diem, Sablatnig; ICDAR 2013), a
contemporary handwriting dataset of 311 writers.

Dataset: https://cvl.tuwien.ac.at/research/cvl-databases/an-off-line-database-for-writer-retrieval-writer-identification-and-word-spotting/

## Contents

- `Writer Identification/` : closed-set identification split (train.txt, test.txt),
  covering 310 writers.
- `Writer Retrieval/` : see the README inside for the dataset link.

Filenames follow `writerid-pageid-lineid-wordid-transcription.tif`, so the writer
id is the part before the first hyphen.

The CVL Database is released under a Creative Commons Attribution-NonCommercial
3.0 licence; please cite Kleber et al. (2013). See the top-level README for more.
