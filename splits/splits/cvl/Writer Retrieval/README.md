# CVL Writer Retrieval

This folder is reserved for the CVL writer retrieval split. The CVL Database was
designed for writer retrieval as well as identification: of its 311 writers, 27
wrote all 7 texts and 284 wrote 5 texts, which gives a natural open-set division.

Original dataset:
https://cvl.tuwien.ac.at/research/cvl-databases/an-off-line-database-for-writer-retrieval-writer-identification-and-word-spotting/

CVL Database, Kleber, Fiel, Diem and Sablatnig, "CVL-Database: An Off-line
Database for Writer Retrieval, Writer Identification and Word Spotting", ICDAR
2013. Released under a Creative Commons Attribution-NonCommercial 3.0 licence.
